# Simple Rails 5 Demo app with React

## Getting Started
Just register with a new user and add a package in the dashboard

###Gems that i use:

*`gem 'devise'`
*`gem 'bootstrap-sass', '~> 3.3.6'`
*`gem 'react-rails'`

###To run
```sh
$ rake db:create
$ rake db:migrate
$ rails s
```
